import { Card, CardHeader, CardTitle, CardDescription, CardContent, CardFooter } from '@kairo-ui/react';

## การใช้งาน

```tsx
import { Card, CardHeader, CardTitle, CardDescription, CardContent, CardFooter } from '@kairo-ui/react';

export function Example() {
  return (
    <Card>
      <CardHeader>
        <CardTitle>Project Alpha</CardTitle>
        <CardDescription>A short summary of the project.</CardDescription>
      </CardHeader>
      <CardContent>
        <p>Card body content goes here.</p>
      </CardContent>
      <CardFooter>
        <span>Updated 2 days ago</span>
      </CardFooter>
    </Card>
  );
}
```

## Props

แต่ละส่วนจะส่งต่อ `ref` ไปยังอีลีเมนต์ดั้งเดิมของมัน รวม `className` ที่กำหนด
เองเข้ากับคลาสพื้นฐาน `kairo-card-*` ของตัวเอง และส่งต่อ native props อื่น ๆ
ทั้งหมดไปด้วย

| Component          | Underlying element | Base class               |
| ------------------- | -------------------- | --------------------------- |
| `Card`              | `<div>`              | `kairo-card`                |
| `CardHeader`        | `<div>`              | `kairo-card-header`         |
| `CardTitle`         | `<h3>`               | `kairo-card-title`          |
| `CardDescription`   | `<p>`                | `kairo-card-description`    |
| `CardContent`       | `<div>`              | `kairo-card-content`        |
| `CardFooter`        | `<div>`              | `kairo-card-footer`         |

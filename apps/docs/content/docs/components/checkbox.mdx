## Usage

```tsx
import { Checkbox } from '@kairo-ui/react';

export function Example() {
  return <Checkbox defaultChecked aria-label="Accept terms" />;
}
```

## Props

| Prop              | Type                                                                                 | Default | Description                                                       |
| ----------------- | ------------------------------------------------------------------------------------- | ------- | --------------------------------------------------------------------- |
| `checked`         | `boolean`                                                                              | —       | Controlled checked state. Use with `onCheckedChange`.                  |
| `defaultChecked`  | `boolean`                                                                               | `false` | Initial checked state for an uncontrolled checkbox.                    |
| `indeterminate`   | `boolean`                                                                               | `false` | Whether the checkbox is in a mixed state (neither checked nor unchecked). |
| `onCheckedChange` | `(checked: boolean, eventDetails) => void`                                             | —       | Called when the checkbox is ticked or unticked.                        |
| `disabled`        | `boolean`                                                                               | `false` | Prevents user interaction.                                             |
| `name`            | `string`                                                                                | —       | Identifies the field when a form is submitted.                        |
| `value`           | `string`                                                                                | —       | The value submitted with the form when the checkbox is checked.       |
| `className`       | `string`                                                                                | —       | Merged with the base `kairo-checkbox` class.                          |
| ...               | Additional props | —       | All other props (e.g. `required`, `readOnly`) are forwarded to the underlying element. |

---
title: Input
description: A native text input styled with Kairo's theme, supporting disabled and invalid states.
---

import { Input } from '@kairo-ui/react';

## Usage

```tsx
import { Input } from '@kairo-ui/react';

export function Example() {
  return <Input placeholder="Email address" aria-label="Email address" />;
}
```

## Props

| Prop           | Type                                     | Default   | Description                                                       |
| -------------- | ------------------------------------------ | ---------- | --------------------------------------------------------------------- |
| `type`         | `string`                                   | `'text'`   | Native `<input>` `type` attribute (e.g. `'email'`, `'password'`).      |
| `aria-invalid` | `boolean \| 'true' \| 'false'`             | —          | Marks the input invalid; styled with a danger border.                 |
| `disabled`     | `boolean`                                   | —          | Prevents user interaction; styled with reduced opacity.               |
| `className`    | `string`                                    | —          | Merged with the base `kairo-input` class.                             |
| ...            | `InputHTMLAttributes<HTMLInputElement>`     | —          | All other native `<input>` props (e.g. `value`, `onChange`, `placeholder`) are passed through. |

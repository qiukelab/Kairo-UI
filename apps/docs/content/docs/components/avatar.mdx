## Usage

```tsx
import { Avatar } from '@kairo-ui/react';

export function Example() {
  return <Avatar src="/jane.png" alt="Jane Doe" fallback="JD" />;
}
```

## Props

| Prop        | Type                       | Default | Description                                                                             |
| ----------- | -------------------------- | ------- | ----------------------------------------------------------------------------------------- |
| `src`       | `string`                   | —       | The image source URL.                                                                      |
| `alt`       | `string`                   | —       | Accessible alt text describing the image.                                                  |
| `fallback`  | `ReactNode`                | —       | Rendered when there's no `src`, or the image fails/hasn't finished loading (e.g. initials). |
| `size`      | `'sm' \| 'md' \| 'lg'`     | `'md'`  | Size of the avatar.                                                                         |
| `className` | `string`                   | —       | Merged with the base `kairo-avatar` class.                                                  |
| ...         | Additional props | — | All other props are forwarded to the underlying element. |

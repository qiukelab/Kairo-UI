---
title: Badge
description: A small label for statuses, counts and metadata, with solid, soft and outline variants across five intents.
---

import { Badge } from '@kairo-ui/react';

## Usage

```tsx
import { Badge } from '@kairo-ui/react';

export function Example() {
  return <Badge intent="success">Active</Badge>;
}
```

## Props

| Prop        | Type                                                              | Default     | Description                             |
| ----------- | -------------------------------------------------------------------- | ------------ | ------------------------------------------ |
| `variant`   | `'solid' \| 'soft' \| 'outline'`                                    | `'solid'`    | Visual style of the badge.                 |
| `intent`    | `'default' \| 'primary' \| 'success' \| 'warning' \| 'danger'`       | `'default'`  | Semantic color of the badge.               |
| `className` | `string`                                                             | —            | Merged with the base `kairo-badge` class.  |
| ...         | `HTMLAttributes<HTMLSpanElement>`                                    | —            | All other native `<span>` props are passed through. |
